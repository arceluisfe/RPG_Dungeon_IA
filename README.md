# ⚔️ RPG Dungeon IA - .NET Edition

¡Bienvenido al Calabozo! Este es un simulador de combate RPG desarrollado en **C#**, enfocado en demostrar el uso de patrones de diseño, asincronía y conectividad con servicios de Inteligencia Artificial.

## 🚀 Características Principales
* **Arquitectura Limpia**: Separación de responsabilidades en diferentes archivos (`Program`, `Entidades`, `ServicioIA`).
* **Patrón Factory**: Generación aleatoria de enemigos (Esqueletos, Orcos, Bugs de Memoria) con estadísticas dinámicas.
* **Narrativa con IA**: Integración de un servicio asíncrono que genera relatos de victoria personalizados basados en el desempeño del combate.
* **Manejo de JSON**: Serialización y deserialización de objetos complejos usando `Newtonsoft.Json`.

## 🛠️ Tecnologías Utilizadas
* **Lenguaje:** C# (.NET 8)
* **Librerías:** Newtonsoft.Json
* **Herramientas:** Git, VS Code

## 🎮 Cómo Ejecutarlo
1. Clona el repositorio:
   `git clone https://github.com/arceluisfe/RPG_Dungeon_IA.git`
2. Navega a la carpeta del proyecto.
3. Ejecuta el comando:
   `dotnet run`
